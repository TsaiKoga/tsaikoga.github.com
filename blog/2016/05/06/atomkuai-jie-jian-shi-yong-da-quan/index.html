快捷键标识

⌘ : Command key

⌃ : Control key

⌫ : Delete key

← : Left arrow key

→ : Right arrow key

↑ : Up arrow key

↓ : Down arrow key

⌥ : Option or Alt key

↩ : Return or Enter key

⇧ : Shift key
